# ⚡ EnergyIQ - Electricity Consumption Analyzer

EnergyIQ is a smart web application that helps users track electricity consumption, estimate monthly bills, analyze appliance usage, forecast future consumption, and receive personalized energy-saving recommendations.

## Features

- Appliance Tracking
- Electricity Bill Calculator
- State-wise Tariff Support
- Monthly Consumption Analytics
- Usage Forecasting
- Anomaly Detection
- Personalized Energy Saving Tips
- Dark / Light Theme
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- Local Storage
